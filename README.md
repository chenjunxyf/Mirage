##Mirage

![logo](logo.ico)

###这是一个用于团队管理的开源项目
    
###主要功能：
+ __周报管理__
+ __团队成员管理__
+ __登录/注册__

###主要技术
+ __后台：nodejs+handlebars摸板__
+ __前端：html、js、css__
+ __框架：express、bootstrap3.2、jquery-1.11.1__
